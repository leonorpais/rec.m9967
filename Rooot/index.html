Introdução & Template inicial

Esta página é uma reconstrução interativa do manual "Bootstrap 5 — Guia Rápido para Iniciantes". Usa o template básico com CSS, Icons e bundle JS via CDN — exatamente como o manual recomenda. :contentReference[oaicite:7]{index=7}

Template mínimo (citado no manual)
<!doctype html>
<html lang="pt-BR">
 <head>
   <meta charset="utf-8">
   <meta name="viewport" content="width=device-width,initial-scale=1">
   <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.1.3/dist/css/bootstrap.min.css" rel="stylesheet">
   <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.8.1/font/bootstrap-icons.css">
 </head>
 <body>
   <!-- seu conteúdo -->
   <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.1.3/dist/js/bootstrap.bundle.min.js"></script>
 </body>
</html>

Salve este ficheiro como HTML e abra no navegador. (Template baseado no manual.)

Layout & Sistema de grade (Grid)

O sistema de grid divide a linha em 12 colunas — use .container, .row e .col-*. A soma dentro de uma .row deve ser até 12 para evitar quebras inesperadas. (Resumo do manual.) :contentReference[oaicite:9]{index=9}

Breakpoints

xs <576px, sm ≥576px, md ≥768px, lg ≥992px, xl ≥1200px, xxl ≥1400px — use classes como col-md-6 para comportamento responsivo.

Gutters (calhas) e espaçamento

Use classes g-* para espaçamento entre colunas (g-0.. g-5). Para margin/padding use utilitários m/p + posição + nível (ex.: mt-3, px-4). :contentReference[oaicite:10]{index=10}

Imagens e figuras

Classes úteis: img-fluid (responsiva), img-thumbnail, rounded, e elementos <figure> com .figure, .figure-img e .figure-caption. :contentReference[oaicite:12]{index=12}

Posicione imagens com float-start, float-end ou centralize com d-block mx-auto.

Tabelas

Adicione table, table-striped, table-hover, table-bordered, e use .table-responsive para tornar a tabela rolável em telas pequenas. (Manual mostra exemplos.) :contentReference[oaicite:13]{index=13}
